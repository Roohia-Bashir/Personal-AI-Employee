# Twitter/X Post Structure & Templates

## Post Anatomy
```
[Hook — 1 punchy line, bold claim, emoji, or surprising fact]
[Context — what happened / what you built, 1-2 lines]
[Value — key insight or result, 1 line]
#Tag1 #Tag2 #Tag3
```

## Hook Patterns by Post Type

### Achievement/Milestone
```
🚀 Just hit [result] in [timeframe].
```

### Build in Public
```
What we shipped this week with AI 🤖
```

### Lesson Learned
```
Made a mistake that cost me [outcome]. Here's the fix:
```

### Insight/Tip
```
Most [role] waste [X hours] on [task]. Here's how to stop:
```

### Announcement
```
🎉 [Feature] is live! [One-line value prop]
```

## Character Count Guide
| Section | Target |
|---|---|
| Hook | 40-60 chars |
| Context + Value | 80-120 chars |
| Hashtags | 40-60 chars |
| **Total** | **< 250 chars** |
