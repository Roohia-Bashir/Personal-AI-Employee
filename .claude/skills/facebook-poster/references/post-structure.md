# Facebook Post Structure & Templates

## Post Anatomy
```
[Hook — 1-2 warm, relatable lines or exciting announcement]
[Context — what happened / what you built, 2-3 lines]
[Value — what this means for your audience, 1-2 lines]
[CTA — question or reaction prompt, 1 line]
#Tag1 #Tag2 #Tag3
```

## Hook Patterns by Post Type

### Achievement/Milestone
```
Big news! We just [achieved result] 🎉
Here's how we did it...
```

### Build in Public
```
We've been building something exciting behind the scenes 🤖
This week we finally shipped it.
```

### Community Engagement
```
Quick question for our community 👇
[Question that invites opinions or experiences]
```

### Tip/Value Post
```
💡 Here's something most people don't know about [topic]:
[Insight in 2-3 lines]
```

### Weekly Update
```
Weekly update from the AI Employee project 📊
Here's what happened this week...
```

## Character Count Guide
| Section | Target |
|---|---|
| Hook | 60-100 chars |
| Context + Value | 150-300 chars |
| CTA | 60-80 chars |
| Hashtags | 40-80 chars |
| **Total** | **300-500 chars optimal** |

## Hashtag Sets by Topic
**AI/Automation:** `#AI #Automation #BuildInPublic #WaheedAI #ClaudeCode`
**Entrepreneurship:** `#Entrepreneur #SmallBusiness #Founder #Productivity`
**Tech:** `#Python #OpenSource #LLM #AIAgents #TechPakistan`

## Tone Guide
- **Warm and conversational** — write like you're talking to a friend
- **Community-first** — invite reactions, comments, and shares
- **Authentic** — share real results and real struggles
- **Encouraging** — Facebook audience responds to positivity
